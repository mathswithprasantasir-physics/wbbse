# WBBSE Class 10 Science Question Bank

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

A comprehensive question bank web application for WBBSE (West Bengal Board of Secondary Education) Class 10 students covering Physics, Chemistry, and Biology.

## 🚀 Live Demo
[View Live Application](#) - *Add your Render URL here after deployment*

## ✨ Features

- **Multi-subject support**: Physics, Chemistry, and Biology
- **Multiple question types**: 
  - Multiple Choice Questions (MCQs)
  - True/False
  - Fill in the Blanks
  - 1 Mark Questions
  - 2 Marks Questions
  - 3 Marks Questions
- **Advanced filtering**: By subject, chapter, question type, marks, and topic
- **LaTeX support**: Mathematical formulas rendered beautifully
- **Export functionality**: Download questions as JSON
- **Responsive design**: Works on all devices
- **Statistics dashboard**: Track question bank progress
- **Show/Hide Answers**: Self-assessment feature
